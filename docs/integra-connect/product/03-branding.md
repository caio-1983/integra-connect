# 03 - Branding

# Branding do Integra Connect

## Posicionamento

O Integra Connect é um produto da Integra Solutions.

Sua identidade comunica organização, confiança e eficiência operacional.

Não buscamos parecer uma empresa de Inteligência Artificial.

Buscamos parecer a empresa que resolve problemas operacionais utilizando
tecnologia como ferramenta.

------------------------------------------------------------------------

# Personalidade da Marca

-   Profissional
-   Moderna
-   Objetiva
-   Confiável
-   Próxima do cliente
-   Elegante sem excessos

------------------------------------------------------------------------

# Valores percebidos

Ao utilizar o produto o cliente deve perceber:

-   organização;
-   produtividade;
-   estabilidade;
-   controle;
-   simplicidade.

Nunca deve perceber complexidade ou excesso de tecnologia.

------------------------------------------------------------------------

# Linguagem

A comunicação deve ser simples e corporativa.

Preferir:

-   Cliente
-   Contato
-   Atendimento
-   Compromisso
-   Oportunidade
-   Operações

Evitar:

-   Lead (quando não necessário)
-   IA como protagonista
-   Jargões técnicos
-   Termos excessivamente "marketizados"

------------------------------------------------------------------------

# Identidade Visual

## Cores

Predominância de fundos claros.

Uso moderado da cor institucional para destacar ações primárias.

Alertas devem utilizar cores semânticas.

Evitar excesso de gradientes e efeitos neon.

------------------------------------------------------------------------

## Tipografia

Hierarquia clara.

Poucos tamanhos.

Leitura confortável.

------------------------------------------------------------------------

## Ícones

Lucide Icons como padrão.

Ícones sempre complementam o texto.

Nunca substituem a descrição.

------------------------------------------------------------------------

# Experiência

A interface deve transmitir:

-   ambiente corporativo;
-   rapidez;
-   organização;
-   previsibilidade.

Cada tela deve responder claramente:

-   Onde estou?
-   O que aconteceu?
-   O que preciso fazer agora?

------------------------------------------------------------------------

# Marca

A marca Integra Solutions permanece presente, porém discreta.

O destaque principal é sempre a identidade da empresa cliente.

A assinatura "Powered by Integra Solutions" pode aparecer em contextos
institucionais, nunca competindo visualmente com o cliente.

------------------------------------------------------------------------

# Princípio Permanente

O branding deve reforçar a percepção de que o Integra Connect é uma
plataforma de operações e relacionamento, não apenas uma ferramenta de
mensagens.
