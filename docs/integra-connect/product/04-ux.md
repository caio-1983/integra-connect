# 04 - UX

# Princípios de UX

A experiência do Integra Connect deve reduzir carga cognitiva, aumentar
produtividade e transmitir controle operacional.

------------------------------------------------------------------------

# Objetivos

-   Reduzir o número de decisões por tela.
-   Tornar o próximo passo sempre evidente.
-   Priorizar ações frequentes.
-   Evitar interrupções desnecessárias.

------------------------------------------------------------------------

# Navegação

A navegação principal representa módulos de negócio, não tecnologias.

Exemplo:

-   Operações
-   Atendimento
-   Clientes
-   Comercial
-   Analytics
-   Administração

------------------------------------------------------------------------

# Workspace

O Workspace é o centro da operação.

Ele deve responder:

-   Quem está falando?
-   O que aconteceu?
-   O que preciso fazer agora?

Toda ação frequente deve estar visível.

------------------------------------------------------------------------

# Hierarquia Visual

1.  Cabeçalho
2.  Ações principais
3.  Conteúdo principal
4.  Informações complementares

------------------------------------------------------------------------

# Componentes

## Sidebar

Responsável apenas pela navegação global.

## Workspace

Área principal de trabalho.

## Painel Lateral

Contexto do cliente.

Nunca deve competir com a conversa.

------------------------------------------------------------------------

# Empty States

Todo estado vazio deve orientar o usuário.

Nunca apresentar apenas "Sem dados".

Exemplo:

"Os compromissos deste cliente aparecerão aqui."

------------------------------------------------------------------------

# Linguagem

Utilizar linguagem objetiva.

Preferir verbos de ação:

-   Criar compromisso
-   Transferir atendimento
-   Registrar oportunidade

Evitar termos técnicos.

------------------------------------------------------------------------

# Feedback

Toda ação deve produzir feedback imediato.

-   sucesso;
-   erro;
-   carregamento;
-   processamento.

------------------------------------------------------------------------

# Acessibilidade

-   Contraste adequado.
-   Ícones acompanhados de texto.
-   Navegação consistente.
-   Componentes previsíveis.

------------------------------------------------------------------------

# Regra Permanente

A interface deve ajudar o usuário a trabalhar.

Nunca deve exigir que o usuário descubra como trabalhar.
