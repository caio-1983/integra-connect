# Regras para Prompts do Claude Code

Todo prompt deve:

- possuir escopo bem definido;
- limitar arquivos quando possível;
- não permitir melhorias fora do escopo;
- preservar arquitetura;
- reutilizar componentes;
- evitar dependências novas.

Ao final informar:

- arquivos alterados;
- componentes reutilizados;
- componentes criados;
- possíveis impactos.
