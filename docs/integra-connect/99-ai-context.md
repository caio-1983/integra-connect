# AI Context

Versão 1.0

Este documento é o contexto padrão para qualquer Inteligência Artificial que trabalhe neste projeto.

Seu objetivo é resumir toda a documentação do produto.

Sempre consulte este documento antes de realizar qualquer implementação.

Caso exista conflito entre documentos, prevalece este documento e as decisões registradas em 06-decisoes.md.

---

# Missão

Transformar uma plataforma existente em um produto SaaS proprietário da Integra Solutions.

O objetivo NÃO é customizar um sistema.

O objetivo é construir um produto nacional.

---

# O Produto

Nome oficial:

Integra Connect

O Integra Connect é uma plataforma Omnichannel de Comunicação Empresarial.

O produto centraliza:

• múltiplos números de WhatsApp

• Instagram

• Facebook Messenger

• Telegram

• Webchat

• E-mail (roadmap)

• futuros canais

Todos os canais deverão funcionar dentro da mesma experiência.

---
